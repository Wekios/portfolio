This is a Belgrade based app, which publish crowd-sourced reviews about local clubs and venues. With it's RESTful routing it's perfectly adapted to CRUD applications. It uses the MEN stack (mongoDb, Express.js, Node.js). Latest addition to the app would be the implementation of the live chat bot. Check it out, and leave a review, BG Showroom.

Website is connected to mongoDB, in which it saves reviews, comments, and user profiles
